Picture Printer - Instructions

Uploading images

You can upload images by clicking in the + icon in the centre of an empty placeholder. Select your image from the dialog that opens and it should appear

Linking from online

You can drag and drop any image from online. You need to be sure that the image you are dragging over is the source, however. If you drag an image that isn't the base URL an alert will pop-up. When this happens you can usually right-click the image you want and "open in a new tab". Drag this image over

You can use the image below to try it out

Removing images

Images should be persistent once added. This means you can open new tabs, close the extension, etc, without losing the images. To remove them all you need to do is click. To get it back you will need to repeat the process how you put it there originally.

Rotating Images

There are currently two ways of dealing with vertical pictures:

  • Rotate the image - the image will be rotated within the placeholder
  • Rotate the placeholder (avoid) - the placeholder will be rotated to fit the image. This currently does not extend to printing.

This seting is global and will ultimitely apply to all images.

Printing

There is no API for the Chrome Printer Page so you will need to marry up the settings:

  • Remove Margins
  • Add background graphics if you want black bars
  • Set the page size
  • Set the page orientation
